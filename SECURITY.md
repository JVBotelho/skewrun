# Security Policy

## Supported Versions

Currently, only the latest release of `skewrun` is actively supported with security updates.

## Reporting a Vulnerability

We take the security of `skewrun` seriously. If you discover a security vulnerability, please do NOT open a public issue.

Instead, please report it privately by sending an email to: **zmtsn3xg.refining218@passinbox.com**

Please include the following information in your report:
- A description of the vulnerability and its impact.
- Steps to reproduce the issue.
- Any potential mitigation or proposed fix if you have one.

We will acknowledge your report within 14 days and provide status updates as we investigate and resolve the issue. We ask that you maintain confidentiality until we have published a fix.
